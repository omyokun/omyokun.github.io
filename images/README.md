# Images Folder

Place your images here, such as:

- `profile.jpg` - Your profile picture (recommended size: 500x500px)
- `research-*.jpg` - Research project images
- `favicon.ico` - Website favicon (optional)
- Any other images for your website

## Image Optimization Tips

- **Profile Picture**: 500x500px, JPG or PNG, under 200KB
- **Research Images**: 800x600px or similar, under 300KB
- Use tools like [TinyPNG](https://tinypng.com/) to compress images
- Consider using WebP format for better compression

## Recommended Formats

- Profile photos: JPG (better for photos)
- Graphics/logos: PNG (better for graphics with transparency)
- Modern browsers: WebP (best compression)
